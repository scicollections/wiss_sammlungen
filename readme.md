# Wissenschaftliche Sammlungen

"Wissenschaftliche Sammlungen" is a Ruby on Rails graph-based documentation system and can be found at https://portal.wissenschaftliche-sammlungen.de/. 

Its development by the [Coordination Centre for Scientific University Collections in Germany](https://wissenschaftliche-sammlungen.de/en/) at Humboldt-Universit√§t zu Berlin was funded by the German [Federal Ministry of Education and Research](https://www.bmbf.de/bmbf/en/home/home_node.html). 

The code is published as free software under the terms of the MIT license. 

## Documentation

The documention can be generated via [yard](http://yardoc.org): Run `yard doc` in the repository root. The options for this command are specified in the `.yardopts` file. The static doc files are written to `public/doc`, the start page is `public/doc/index.html`. 

The current online version of this documentation with further information about requirements and application setup can be found at https://portal.wissenschaftliche-sammlungen.de/doc/.
